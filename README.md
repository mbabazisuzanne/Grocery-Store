### Endpoints
- /auth/signup `for adding a user`
- /auth/view `to View users`
- /inventory `to view entire stock`
- /inventory/item `to add item to inventory`
- /inventory/items `to view all the items `

### Item Schema
title:{type:text},
stock:{type:number},
price:{type:number},
category:{type:number}

### User Schema
